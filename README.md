# EV Charging Behavior Analysis

Multi-task machine learning pipeline on an electric vehicle charging dataset.

## Tasks
- Cost prediction (regression)
- Charging time prediction (regression)  
- Long session classification

## Methods
- Data preprocessing with sklearn pipelines
- GridSearchCV hyperparameter tuning
- Anomaly detection: Isolation Forest + One-Class SVM
- Model explainability: SHAP + LIME

## Tools
Python, scikit-learn, pandas, NumPy, SHAP, LIME, joblib
